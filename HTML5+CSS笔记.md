# HTML5笔记

## 一）XHTML文档结构

```html
<!--［XHTML 文档基本框架］-->

<!-- 定义 XHTML 文档类型-->
< ! DOCTYPE html PUBLIC  "-//W3C//DTD XHTMLl.0 Transitional//EN”  "http://www.w3.org/TR/xhtml1/DTD/xhtmll-transitional.dtd">

<!--XHTML 文档根元素，其中 xmlns 属性声明文档命名空间-->
<html xmlns=”http://www.w3.org/1999/xhtml " >

<!--头部信息结构元素-->
<head>
<!--设置文档字符编码-->
<meta http- equiv=”Content-Type" content=”text/html; charset=gb2312”/>
<!--设置文档标题-->
<title>无标题文档</title>
</head>

<!--主体内容结构元素-->
<body></body>
</html>  
```
