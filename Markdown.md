Markdown 使用反斜杠转义特殊字符：

Markdown 支持以下这些符号前面加上反斜杠来帮助插入普通的符号：  
```
\   反斜线
`   反引号
*   星号
_   下划线
{}  花括号
[]  方括号
()  小括号
'#'   井字号
+   加号
-   减号
.   英文句点
!   感叹号
```
